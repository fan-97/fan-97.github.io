java集合框架

2019-07-28 java基础学习总结

java.util包下的集合框架

四大集合接口

  • Collection

    Collection 层次结构 中的根接口。Collection 表示一组对象，这些对象也称为 collection 的元素。

  • List
  • Set
  • Map

List

有序可重复Collection集合 可以拥有多个null元素

常用的方法：
  • boolean add(E e) 添加元素到列表的尾部
  • void add(int index, E element) 向指定位置添加元素
  • E get(int index) 返回列表指定位置的元素
  • E remove(int index) 移除指定位置的元素

常用子类：

  • ArrayList:

  • LinkedList

里面的变量
transient Node first;
transient Node last;

transient:关键字 当序列化的时候忽略这个关键字修饰的变量 不能放在方法体类，只能用来修饰成员变量或者类变量

private static class Node {
E item;
Node next;
Node prev;

Node(Node prev, E element, Node next) {
this.item = element;
this.next = next;
this.prev = prev;
}
}

LinkedList是对链表进行操作，底层是链表实现，还实现了Deque 可以进行栈对应的操作

pop() 将栈顶元素出栈并移除该元素
push(E) 入栈，采用的是头插法
poll() 将栈顶元素出栈但不移除元素
↑ 方法是ArrayList没有的方法

<font color=    #A52A2A>优点：插入，删除元素快，只需要删除对应的结点，不需要移动元素
缺点：查找元素慢，因为需要从头结点遍历  可以采用双向链表优化</font>

#### Set
> 无序不可能重复的Collection集合 最多只能拥有一个null 元素 
> 底层是Map存放key,value键值对，因为Key不能重复，所以set不能存放重复的元素

常用方法(Set接口中的方法):

add(E e) : 添加一个set集合中没有的元素 添加成功返回true
contains(Object o) : 查询是否集合中是否包含该元素
isEmpty() : 为空返回true
iterator() : Set集合的迭代器
remove(Object o) : 移除某个元素
size() : 集合的大小

<font color=    #FF0000>常用子类</font>
* HashSet
> 由哈希表（实际上是一个 HashMap 实例）支持。它不保证 set 的迭代顺序；特别是它不保证该顺序恒久不变。此类允许使用 null 元素。**线程不同步**

剖析源码：

private transient HashMap<E,Object> map;
private static final Object PRESENT = new Object();

public HashSet() {
map = new HashMap<>();
}

public boolean add(E e) {
return map.put(e, PRESENT)==null;
}

public boolean remove(Object o) {
return map.remove(o)==PRESENT;
}

<font color=#000FFF>
总结：
1.hashSet底层实现是 hashmap  对set的一切操作实质都是在操作map 
2.之所以set存放的元素是不重复的，从源码可以知道 当向set中add一个元素时，是在向map中put一个键值对，key 就是参数 e  value 是一个固定的object对象 所有的value都是一个值。而map中 key又是唯一的，当有相同的key 被put的时候，会覆盖掉之前的key 。所以当add一个已经存在的元素时，map.put()返回的是旧的value值。
3.remove同理。因为value值都是固定的Object 且是final修饰的，不会增加额外的内存开销。
</font>

* LinkedHashSet
> 是HashSet的子类，但是维护了一个双向链表，所以add remove等操作更高效

--------
* TreeSet
> 基于 TreeMap 的 NavigableSet 实现。使用元素的自然顺序对元素进行排序，或者根据创建 set 时提供的 Comparator 进行排序，具体取决于使用的构造方法。
